## 2. Design System — 设计规范

### 2.1 设计原则

| 原则 | 说明 | 应用示例 |
|------|------|---------|
| **内容优先** | 页面服务于内容，装饰服务于阅读 | 文章详情页最大宽度 720px，无干扰侧边栏 |
| **极简克制** | 每个元素都有存在的理由，无装饰性冗余 | 白色背景 + 灰色辅助线，拒绝花哨渐变 |
| **一致性** | 同一交互在不同页面表现一致 | 所有页面使用相同导航、相同按钮样式 |
| **可访问** | 所有人都能使用，包括视觉障碍者 | WCAG 2.1 AA 级色彩对比度 |
| **渐进披露** | 复杂功能不一次性展示，按需展开 | 发布面板用侧边抽屉，而非嵌入编辑器 |

### 2.2 色彩系统

### 2.7 图标系统

| 属性 | 规范 |
|------|------|
| **图标库** | Material Icons (Outlined 变体为主) |
| **默认尺寸** | 20px（导航图标 24px） |
| **颜色** | 继承文字颜色（currentColor） |
| **间距** | 图标与文字间距 8px |

### 2.8 按钮规范

#### 按钮层级

| 层级 | 样式 | 使用场景 |
|------|------|---------|
| **Primary** | 实心蓝底白字 `#2563EB` + `radius-md` + `shadow-sm` | 主操作（发布、提交、保存） |
| **Secondary** | 白底蓝边蓝字，边框 1px `#2563EB` | 次操作（取消发布、编辑） |
| **Tertiary** | 白底灰边灰字，边框 1px `#E5E7EB` | 辅助操作（返回、取消） |
| **Ghost** | 无边框无背景，仅文字 | 最轻量操作（链接式按钮） |
| **Danger** | 实心红底白字 `#EF4444` | 破坏性操作（删除、冻结） |
| **Icon** | 仅图标无文字 | 工具栏按钮 |

#### 按钮尺寸

| 尺寸 | 高度 | Padding | 字号 | 用途 |
|------|------|---------|------|------|
| `sm` | 32px | 8px 12px | 14px | 紧凑场景（表格操作） |
| `md` | 40px | 12px 16px | 16px | 默认尺寸（表单提交） |
| `lg` | 48px | 16px 24px | 16px | 突出操作（发布按钮） |

#### 按钮状态

| 状态 | 表现 |
|------|------|
| **Default** | 如上定义 |
| **Hover** | 背景色加深 10%（Primary→`#1D4ED8`），加 `shadow-md` |
| **Pressed** | 背景色再加深 10%，缩至 `shadow-sm` |
| **Disabled** | 灰色背景 `#F3F4F6`，灰色文字 `#9CA3AF`，`opacity: 0.6` |
| **Loading** | 禁用态 + 左侧旋转 Spinner，文字变"处理中..." |

### 2.9 输入框规范

| 属性 | 规范 |
|------|------|
| **高度** | 40px（md），48px（lg） |
| **边框** | 1px `#E5E7EB`，聚焦时 2px `#2563EB` |
| **圆角** | `radius-md` (8px) |
| **背景** | `#FFFFFF`（默认），`#F9FAFB`（禁用） |
| **内边距** | 12px 16px |
| **字号** | 16px（防止 iOS 自动缩放） |
| **Label** | 上方 14px `text-secondary`，间距 8px |
| **Placeholder** | `#9CA3AF` `text-sm` |
| **Error** | 边框变红 `#EF4444`，下方红色错误文字 12px |
| **Success** | 边框变绿 `#10B981`，右侧绿色勾图标 |

### 2.10 卡片规范

| 属性 | 规范 |
|------|------|
| **背景** | `#FFFFFF` |
| **圆角** | `radius-lg` (12px) |
| **阴影** | `shadow-sm`（默认），`shadow-md`（Hover） |
| **内边距** | `space-4` (16px) |
| **边框** | 1px `#E5E7EB`（可选，默认无） |
| **Hover** | Y 轴上移 2px + `shadow-md` + 0.2s ease |

### 2.11 响应式断点

| Token | 值 | 设备 | 布局策略 |
|-------|------|------|---------|
| `xs` | < 375px | iPhone SE | 单列紧凑 |
| `sm` | 375-639px | 手机 | 单列，缩小间距 |
| `md` | 640-1023px | 平板/小屏笔记本 | 双列/单列自适应 |
| `lg` | 1024-1439px | 笔记本 | 双列+侧边栏 |
| `xl` | ≥ 1440px | 大屏 | 三列布局 |

### 3.1 博客首页（文章列表页）

#### 页面目标

> 让读者快速浏览博主所有已发布文章，建立第一印象，引导进入文章详情页。

│ │                ────────  [搜索框] ────────          │ │
│ └──────────────────────────────────────────────────┘ │
│                                                        │
│ ┌──────────────────────────────────────────────────┐ │
│ │  Hero 区（可选）                                   │ │ ← 博主简介卡片
│ │  [头像 80px] 博客名称（24px bold）                  │ │
│ │  "个人简介文字..."                                  │ │
│ │  [GitHub] [Twitter] [RSS]                          │ │
│ └──────────────────────────────────────────────────┘ │
│                                                        │
│ ┌──────────────────────────┐  ┌─────────────────────┐ │
│ │ ┌──────────────────────┐ │  │                     │ │ ← 文章卡片 + 侧边栏
│ │ │ [封面图 16:9]        │ │  │ 分类列表            │ │
│ │ │                      │ │  │  • 技术 (12)        │ │
│ │ │ 文章标题 (20px)      │ │  │  • 生活 (8)         │ │ ← 侧边栏
│ │ │ 摘要文字 (14px, 2行) │ │  │  • 读书 (5)         │ │
│ │ │                      │ │  │                     │ │
│ │ │ [标签] | 分类 | 日期 │ │  │ 标签云              │ │
│ │ │      | 阅读量 | 阅读时间│ │  │  分布式 系统设计 后端 │ │
│ │ └──────────────────────┘ │  │  Python 读书 笔记    │ │
│ │                          │  │                     │ │
│ │ ┌──────────────────────┐ │  │ 归档                │ │

#### 布局（移动端）

```
┌──────────────────────┐
│ [≡] 博客名称  [🔍]   │ ← 简化导航
│ ──────────────────── │
│ ┌──────────────────┐ │
│ │ [封面图 16:9]    │ │ ← 单列卡片流
│ │ 文章标题         │ │
│ │ 摘要(2行)        │ │
│ │ 标签 | 日期      │ │
│ └──────────────────┘ │
│ ┌──────────────────┐ │
│ │ 下一篇文章...    │ │
│ └──────────────────┘ │
│                      │
│ < 1  2  3  >         │ ← 简化分页
└──────────────────────┘
```

#### 核心组件

| 组件 | 说明 |
|------|------|
| **NavBar** | 固定顶部，博客Logo + 导航链接 + 搜索触发按钮 |
| **HeroCard** | 博主信息卡片（头像 + 名称 + 简介 + 社交链接），首页首屏 |
| **ArticleCard** | 文章卡片（封面图 + 标题 + 摘要 + 元信息） |
| **Sidebar** | 分类列表 + 标签云 + 归档（桌面端右侧） |
| **Paginator** | Keyset 分页器（上一页/下一页 + 页码跳转） |
| **SearchTrigger** | 搜索按钮（移动端图标，桌面端搜索框） |

#### 按钮与交互

| 元素 | 类型 | 行为 |
|------|------|------|
| 文章卡片 | 卡片（可点击） | Hover 上浮+阴影加深 → 点击进入文章详情页 |
| 导航链接 | Ghost 按钮 | Hover 文字变蓝，当前页下划线指示 |
| 搜索框 | 输入框 + 图标 | 输入关键词后回车 → 跳转搜索结果页 |
| 分页器 | 页码按钮 | 当前页蓝色高亮，其他页灰色，Hover 变蓝 |
| RSS 图标 | Icon 按钮 | 点击打开 RSS Feed URL |

#### 颜色建议

- 页面背景：`#FFFFFF`
- 卡片背景：`#FFFFFF`（默认态），Hover 时 `shadow-md`
- Hero 区背景：`#F9FAFB`
- 封面图区域：默认占位图使用渐变 `#DBEAFE → #2563EB`
- 标签文字：`#6B7280`，标签背景 `#F3F4F6`
- 分割线：`#E5E7EB`

#### 响应式要点

| 断点 | 变化 |
|------|------|
| ≥ 1024px (lg) | 双列布局（主内容 70% + 侧边栏 30%） |
| 640-1023px (md) | 单列布局，侧边栏移至底部 |
| < 640px (sm) | Hero 卡片简化（隐藏简介文字），卡片封面图缩窄，导航折叠为汉堡菜单 |

## 6. 响应式与移动端适配

### 6.1 全局响应式策略

| 策略 | 说明 |
|------|------|
| **桌面优先** | 先设计桌面端完整布局，再逐步缩减至移动端 |
| **Flexbox/Grid** | 所有布局使用 CSS Flexbox 或 Grid，不使用固定像素宽度 |
| **断点切换** | 通过 Design System 断点 token 切换布局模式 |
| **触摸优化** | 移动端所有可点击元素最小 44px × 44px 触摸区域 |
| **性能优先** | 移动端减少请求量：侧边栏/TOC 默认隐藏，图片懒加载 |

### 6.2 各页面移动端适配要点

| 页面 | 桌面 → 移动端关键变化 |
|------|----------------------|
| **首页** | 双列→单列；侧边栏移至底部"分类/标签"折叠区；导航变汉堡菜单；Hero卡片简化 |
| **文章详情** | 正文字号 16→15px；TOC 隐藏或移至顶部折叠面板；代码块字号 14→13px；作者卡片简化 |
| **分类/标签归档** | 文章网格→单列卡片流；标签云字号缩小 |
| **搜索结果** | 搜索框置顶；结果列表单列 |
| **登录/注册** | 居中卡片占满屏幕宽度（减去 16px padding） |
| **编辑器** | 左右分栏→Tab切换模式（源码/预览）或纯源码模式 |
| **我的文章** | 表格→卡片列表（每篇文章一个卡片） |
| **后台管理** | 侧边栏→底部Tab栏；表格→卡片列表；统计卡片→2列网格 |
| **用户中心** | 单列表单（不变），社交链接编辑器全宽 |

## 8. 无障碍设计（A11Y）

### 8.1 WCAG 2.1 AA 级合规

| 要求 | 实施方案 |
|------|---------|
| **色彩对比度 ≥ 4.5:1** | 所有文字-背景组合满足对比度要求；`#18191C` on `#FFFFFF` = 16.5:1 ✅ |
| **非文字对比度 ≥ 3:1** | 边框、图标、表单控件满足 3:1 |
| **焦点指示** | 所有可交互元素聚焦时显示 2px `#2563EB` 边框 |
| **键盘导航** | Tab 焦点流清晰可见，Escape 关闭模态框/抽屉 |
| **文字不可仅靠颜色** | 错误不只变红色，还加图标和文字说明 |
| **图片 alt 文字** | 所有 `<img>` 含描述性 alt；装饰性图片 alt="" |

### 8.2 语义化 HTML 结构

| 页面区域 | 语义标签 |
|---------|---------|
| 导航栏 | `<nav>` + `aria-label="主导航"` |
| 主内容 | `<main>` |
| 侧边栏 | `<aside>` |
| 文章 | `<article>` |
| 文章标题 | `<h1>` |
| 章节标题 | `<h2>` → `<h3>` 层级递进 |
| 页脚 | `<footer>` |
| 搜索框 | `<search>` 或 `role="search"` |

### 8.3 ARIA 增强

| 组件 | ARIA 属性 |
|------|-----------|
| **Tabs** | `role="tablist"` / `role="tab"` / `role="tabpanel"` + `aria-selected` |
| **模态框** | `role="dialog"` + `aria-modal="true"` + `aria-labelledby` |
| **抽屉** | `role="complementary"` + `aria-label="发布配置"` |
| **进度条** | `role="progressbar"` + `aria-valuenow` / `aria-valuemin` / `aria-valuemax` |
| **Toast** | `role="status"` + `aria-live="polite"` |
| **加载态** | `aria-busy="true"` + `aria-label="加载中"` |
| **表单错误** | `aria-invalid="true"` + `aria-errormessage` |

### 8.4 减弱动画偏好

```css
@media (prefers-reduced-motion: reduce) {
  * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
}
```

尊重用户系统偏好，关闭或极简化所有动画和过渡效果。
